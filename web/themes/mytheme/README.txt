
Steps to integerate Mytheme

1. Dump the sql file in your server 
2. Install fresh drupal8 project in your system with given .sql file
3. Place the mytheme folder inside this path project/web/themes
4. open the project in the browser and login as a admin
5. navigate to admin/appearance/click +install new theme  and upload the zip folder that you have downloaded 
6. install the theme and set as defualt 
7. clear the cache and run the project in browser.
